# hereiam

1996년 PC통신 시절 개인홈페이지 감성으로 만든 홈피입니다.
seoul.craigslist.org에서 영감을 받았어요.

이름 뜻: "여기 내가 있어요"

## 특징

- **순수 HTML**: 프레임워크 없음 (메모장으로 작성)
- **회색 배경 + Times New Roman**: Win95 + Netscape Navigator 시대의 룩
- **deprecated 태그 적극 사용**: `<center>`, `<font>`, `<marquee>`, `<blink>`
- **공사중 박스**: 90년대 홈피의 필수 요소
- **방문자 카운터**: 검은 박스에 lime 글씨
- **웹링**: 친구 홈피로 가는 옛날 방식 네비
- **빠른 로딩**: 이미지 0개, CSS 단일 파일

## 파일 구조
