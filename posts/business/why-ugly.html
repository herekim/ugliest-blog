:: 크레이그리스트 충격사건 ::

몇년전에 처음 seoul.craigslist.org 봤을때 진짜 충격이었어요!!!

2026년인데도 90년대 그때 그모습 그대로!!! HTML 표만 가득하고, 링크는 그냥 파란색이고, 글꼴은 Times New Roman이고, 여백도 거의 없고...

근데 이상하게...

  • 로딩이 진짜 빨라요!!! (1초도 안걸려요)
  • 찾고싶은 정보가 잘 보여요
  • 핸드폰에서도 잘 돌아가요
  • 서버비도 거의 안나올거에요
  • 사람들이 계속 써요

이게 사업의 본질 아닌가 싶더라구요...

:: 디자인 너무 한 함정 ::

YPLabs에서 일하면서 느낀건요...

  1. 디자인 시간이 개발 시간보다 길어져요!!!
    피그마에서 완벽한 디자인 시스템 만든다고 몇주씩 쓰고, 정작 진짜 기능은 뒷전이 되더라구요 ㅠ_ㅠ
  2. 사용자는 예쁜것보다 빠른걸 원해요
    아무리 예쁜 애니메이션도 3초 로딩은 못이겨요
  3. 나중에 고치는게 지옥이에요
    CSS 프레임워크 업데이트 할때마다 화면 깨지고...;;

:: 그래서 못생긴게 좋아요!! ::

  • [속도] CSS 파일 3KB!! 그림도 없어요
  • [접근성] 스크린리더, 옛날 브라우저 다 잘됨
  • [집중] 디자인 신경안쓰고 글에만 집중

  • [오래감] 10년뒤에도 그대로 작동할거에요
